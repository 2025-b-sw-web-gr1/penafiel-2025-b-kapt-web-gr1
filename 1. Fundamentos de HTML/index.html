Fundamentos de HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza "etiquetas" para estructurar el contenido.

Etiquetas básicas:

  • <h1> a <h6>: Encabezados
  • <p>: Párrafos
  • <a>: Enlaces
  • <img>: Imágenes
  • <ul>, <ol>, <li>: Listas
