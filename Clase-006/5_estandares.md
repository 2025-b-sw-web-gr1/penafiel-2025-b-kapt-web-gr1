## HTML
Es un lenguaje de marcado que define la estructura y el contenido de las páginas web, permitiendo describir texto, imágenes y otros objetos de forma que puedan ser interpretados por navegadores web.
## CSS (Cascading Style Sheets)
Es un lenguaje de estilo que permite a los desarrolladores controlar la presentación de páginas web, separando el contenido de su estructura visual.
## DOM (Document Object Model)
Es una interfaz de programación de aplicaciones (API) que establece la estructura lógica de los documentos y el modo en que se accede y manipula su contenido.
## WCAG (Web Content Accessibility Guidelines)
Son un conjunto de estándares y directrices que se basan en cuatro principios fundamentales: Perceptible, Operable, Comprensible y Robusto (POUR), que garantizan que la información y los componentes de la interfaz sean perceptibles, operables, comprensibles y robustos para una amplia variedad de agentes de usuario, incluidas las tecnologías de asistencia.
## Web APIs y Semantic Web standares

APIs Web permiten a los desarrolladores crear elementos personalizados reutilizables para páginas web, facilitando así el desarrollo y mantenimiento de aplicaciones web, con el objetivo de lograr una mayor interoperabilidad entre diferentes navegadores y plataformas.

La web semantica promueve tecnologías que permiten añadir significado a los recursos en la Web, como documentos, imágenes o vídeos, de forma que las máquinas puedan interpretarlos de manera similar a como lo hacen los humanos.
